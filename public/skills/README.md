# 20 Skills Claude AI — Guide d'installation

Ces 20 skills transforment Claude en agents spécialisés pour votre activité.
Chaque fichier `.md` est auto-suffisant : collez-le et Claude devient cet expert.

---

## Sur claude.ai (Projects) — Recommandé

1. Allez sur [claude.ai](https://claude.ai)
2. Cliquez sur **"Projects"** dans la sidebar
3. Créez un nouveau projet ou ouvrez un existant
4. Dans les **"Project Instructions"**, collez le contenu du fichier `.md`
5. Toutes les conversations dans ce projet utiliseront automatiquement le skill

**Astuce :** Créez un projet par domaine — "Emails & Communication", "Stratégie Business", "Organisation" — et collez les skills correspondants.

---

## Sur Claude Desktop

1. Ouvrez Claude Desktop
2. Allez dans **Settings → Skills**
3. Cliquez sur **"Add custom skill"**
4. Collez le contenu du fichier `.md`
5. Le skill est actif dans toutes vos sessions

---

## Avec Claude Code (terminal)

1. À la racine de votre projet, créez ou ouvrez le fichier `CLAUDE.md`
2. Collez le contenu du skill
3. Claude Code le lira automatiquement à chaque session
4. Variante : créez `.claude/skills/nom-du-skill/SKILL.md`
