# 1. L'AGENCE

Aurentia Agency est une agence tech IA. On ne fait pas "juste des sites web". On conçoit des expériences digitales complètes.

## Les 3 métiers

**Métier 1 — Sites vitrines sur-mesure**
Pour les entreprises locales qui veulent exister en ligne. On attaque niche par niche (conciergeries d'abord, puis salles de sport, restaurants, hôtels). Chaque site est unique, zéro template, SEO intégré, livré en 48h. À partir de 900€. C'est l'offre d'entrée, celle qui tourne déjà avec les packs Essentiel / Croissance / Premium.

**Métier 2 — Applications SaaS**
Développement d'applications web complètes : dashboards, portails clients, outils métier, back-offices, plateformes. Notre propre produit "Aurentia for Agencies" est la preuve de ce savoir-faire. Sur devis.

**Métier 3 — Landing pages SaaS**
Pages de conversion haute performance pour des produits tech/SaaS. Design spectaculaire, animations premium, optimisées conversion. À partir de 1 500€.

**IMPORTANT pour la home :** Les packs (Essentiel 900€, Croissance 1490€, Premium 2200€) sont spécifiques à la niche conciergeries. Ils ne doivent PAS apparaître sur la home. La home présente les 3 métiers avec des fourchettes génériques. Le détail des packs par niche va sur les sous-pages (/conciergeries, /restaurants, etc.).

# 4. LE POSITIONNEMENT

## Le problème qu'on résout

La majorité des entreprises ont soit pas de site, soit un site médiocre. Les agences classiques = 3 000-10 000€ et 6-12 semaines. Les freelances = imprévisibles. Les templates = génériques. Résultat : des milliers d'entreprises restent invisibles en ligne.

## Notre réponse

L'IA accélère tout (design, code, SEO, contenu) + 20 ans d'expertise humaine garantissent la qualité. Vitesse d'une IA + craft d'un artisan. Prix imbattables.

## La promesse

"Tant que c'est pas parfait, on ne lance pas."

On utilise l'IA pour amplifier 20 ans d'expertise — pas pour remplacer le craft humain. Chaque projet bénéficie de :

# 7. LE TON DE VOIX

- **Premium + confiant + direct.** On sait ce qu'on vaut.
- **Chiffres > mots** — "48h" pas "rapidement", "900€" pas "prix compétitif"
- **Phrases courtes** — 15 mots max. Direct.
- **Humain** — On parle comme des humains, pas comme un chatbot corporate.
- **Confiant** — On affirme. "On fait X", pas "On peut peut-être vous aider à X".

### Mots à utiliser

révéler, illuminer, propulser, sur-mesure, accessible, forger, potentiel, l'aube, naître, lumière, craft, artisan

### Mots interdits

cheap, automatique, template, basique, robot, garanti

# 12. CONTEXTE TECHNIQUE (depuis CLAUDE.md)

## Stack complète

- **Framework :** Next.js 14+ (App Router)
- **Language :** TypeScript (strict mode)
- **Styling :** Tailwind CSS + shadcn/ui
- **State :** React Query (TanStack Query) + Zustand si besoin
- **Forms :** React Hook Form + Zod
- **Animations :** Framer Motion + GSAP
- **Database :** Supabase (PostgreSQL + Auth + Storage + Realtime)
- **API :** Next.js Server Actions ou Supabase Edge Functions
- **Payments :** Stripe
- **Automation :** N8N pour les workflows
- **Fonts :** Recoleta (headings), Plus Jakarta Sans (body), JetBrains Mono (mono)

## Registries shadcn disponibles

- `@blocks` — Blocks.so
- `@animate-ui` — Animate UI
- `@lucide-animated` — Icônes animées
- `@supabase` — Composants Supabase
- `@bundui` — Sections landing page
- `@billingsdk` — Pricing tables
